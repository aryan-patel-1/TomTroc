# TomTroc

TomTroc est une plateforme web permettant aux utilisateurs **d’échanger des livres entre particuliers**.
Le projet met en avant la simplicité, la convivialité et le partage autour de la lecture.

---

## Fonctionnalités principales

* Authentification des utilisateurs (inscription / connexion)
* Gestion du profil utilisateur (avatar, informations personnelles)
* Ajout, modification et suppression de livres
* Système de messagerie entre utilisateurs
* Consultation et recherche de livres disponibles
* Upload d’images (avatars, livres)

## Installation

1. Cloner le dépôt :

```bash
git clone https://github.com/aryan-patel-1/tomtroc.git
```

2. Configurer la base de données :

* Créer une base MySQL
* Importer le fichier `bdd.sql` fourni
* Renseigner les identifiants dans le fichier de configuration

3. Lancer le projet via un serveur local (MAMP, XAMPP, WAMP)
